    Shortcuts & Commands

    Scheduling

    • /shift — Today’s shift
    • /week — Next 7 days
    • /swap YYYY-MM-DD 12:00-20:00 note — Request swap
    • /swaps — View swap board

    Pay

    • /pay — Today’s estimated pay
    • /nextpay — Next payday

    Learning

    • /policy <term> — Search policies
    • /quiz start — 5-question training quiz
    • /quit — Exit quiz

    Wellbeing

    • /break 20 — Start break timer (mins)
    • /cancelbreak — Stop timer

    Tip: Press Ctrl/⌘+K to open this palette anytime.
